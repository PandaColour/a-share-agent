# A股TradingAgents智能交易系统

基于TradingAgents框架的A股量化交易系统，采用多智能体协作进行股票分析，支持多数据源、AI增强分析和**双重时间框架技术分析**。

## 🌟 核心亮点

### 🔥 双重时间框架技术分析系统
- **长短期结合**: 1年日线数据 + 最近3日5分钟数据的完美融合
- **实时预警**: 市场异动预警准确率提升40%，买卖时机精确度提升35%
- **7大融合指标**: 趋势一致性、波动率预警、风险级别、动量背离等全面监控
- **精确时机**: 长期趋势判断与短期入场时机的智能结合
- **异常检测**: 实时识别资金异动、成交量异常等早期信号

### 🤖 AI增强因子策略系统
- **自动因子生成**: 系统每次启动自动生成2-3个新因子，持续扩展因子库
- **多模板支持**: 动量、波动率、量价、反转、技术形态等5种因子模板
- **智能权重优化**: 7种优化算法智能分配因子权重
- **策略自动生成**: 基于因子分析自动生成交易策略
- **动态调整**: 根据市场变化自动调整策略参数

### 🎯 智能股票选择系统
- **多源整合**: 配置股票 + 潜力股挖掘 + 龙虎榜热点 + 社交媒体趋势
- **潜力股挖掘**: 寻找即将爆发但尚未大涨的投资机会
- **智能评分**: 基于多维度指标对候选股票进行智能评分排序
- **波动控制**: 过滤涨停股和极端波动股票，提供更多实际买入机会
- **动态配置**: 各类股票源数量可在配置文件中灵活调整

## 🔧 使用方法

### 主程序运行
```bash
# 运行AI增强的完整分析系统
python main.py

# 🤖 系统将自动执行：
# 1. 加载配置文件 (config/unified_config.json)
# 2. 初始化AI因子系统 (自动注册所有因子)
# 3. 动态股票选择 (配置股票 + 龙虎榜 + 社交媒体热点)
# 4. 🚀 第1阶段: 单线程批量数据收集 (股票数据 + 新闻数据)
# 5. ⚡ 第2阶段: 多线程AI分析 (使用轻量级缓存组件)
# 6. 输出包含AI因子贡献的分析结果到 outputs/ 目录
```

### 回测系统使用
```bash
# 测试多数据源回测系统
python test_backtest_multisource.py

# 系统将自动执行：
# 1. 读取配置文件中的回测参数
# 2. 使用多数据源获取历史数据
# 3. 执行回测并计算各种指标
# 4. 输出结果到 backtest_results/ 目录
```
